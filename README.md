# 🎨 Rang.Lab — Professional Color Design Studio

**Rang.Lab** is a high-performance color workstation designed for artists, designers, and developers. It moves beyond standard random generators by using mathematical **LCH Perceptual Mapping** to ensure every palette is visually balanced, accessible, and production-ready.

## 🚀 The Vision: Why Rang.Lab?
Most color tools on the web feel like toys—they provide random HEX codes that often clash or fail accessibility tests. **Rang.Lab** was built to bridge the gap between "random colors" and "professional design systems." 

We wanted to create a workspace where a creator can:
* **Generate** with mathematical precision using neural seeds.
* **Extract** inspiration from real-world architecture and nature images.
* **Visualize** work on actual UI mockups before writing a single line of CSS.
* **Organize** assets into cloud-synced folders for real-world client projects.

## 🧠 The Science: Color Logic & Perceptual Math
Unlike standard generators that use RGB or HSL, Rang.Lab's core engine operates in the **LCH (Lightness, Chroma, Hue)** color space.

- **Perceptual Balance:** RGB is for screens, not human eyes. LCH allows us to keep "Lightness" constant across different hues, so a yellow and a blue feel like they have the same visual "weight."
- **Harmonious Algorithms:** The engine follows mathematical rules (Analogous, Monochromatic, and Complementary) to ensure palettes follow design principles.
- **A11y-First Architecture:** Built-in contrast checking against **WCAG 2.1 standards** ensures every design is inclusive and readable.

## 🛠️ The Tech Stack
Built with the MERN stack and optimized for high-end micro-interactions.

* **Frontend:** React.js, Tailwind CSS, GSAP (Animations).
* **Backend:** Node.js, Express.js.
* **Database:** MongoDB Atlas.
* **Security:** JWT Authentication & Google OAuth 2.0.
* **Color Math:** Chroma.js integration.

## ⚡ The Workflow: From Seed to Sync
1.  **Initialize Identity:** Play as a Guest or register as an Artist to unlock cloud storage.
2.  **The Lab:** Use the neural generator to synthesize infinite harmonic variations.
3.  **Refine:** Extract shades, check contrast, and test on live website mockups.
4.  **Sync:** Save architectures into project-specific folders for instant access anywhere.
